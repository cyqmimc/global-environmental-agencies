# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

- `npm run dev` — start Vite dev server
- `npm run build` — production build to `dist/`
- `npm run preview` — preview production build locally
- `npm run fetch-data` — fetch latest World Bank data → `public/wb-data.json` (recommend quarterly)
- `npm run check-updates` — scan all data sources, report what may need updating

No test runner or linter is configured. See `DATA-MAINTENANCE.md` for data update sources and procedures.

## Architecture

Single-page React app: 80 countries, environmental agencies, World Bank data, treaty compliance tracking. Built with Vite 8 + React 18 + Tailwind CSS v4 + Chart.js (Bar + Radar).

**Component structure:**
- `src/App.jsx` — main component: state, data fetching/merging, layout, filters, view toggle
- `src/WorldMap.jsx` — SVG world map with 4-metric coloring + hover/click
- `src/components/DetailDialog.jsx` — country detail popup (scorecard, compliance dashboard, treaties, charts)
- `src/components/CompareDialog.jsx` — 2-3 country comparison table + charts
- `src/components/RankingsView.jsx` — sortable leaderboard table with composite scoring
- `src/components/Scorecard.jsx` — A-F grade report card per country (6 dimensions)
- `src/components/AboutPage.jsx` — data sources, methodology, licenses
- `src/constants.js` — shared constants (TREATY_LABELS, RESPONSIBILITY_LABELS, NDC_RATING_CONFIG, URL helpers, CSV export)

**Data files:**
- `public/countries.json` — 80 countries, curated agency data (manually maintained)
- `public/wb-data.json` — World Bank indicators (auto-generated by fetch script)
- `public/world-map.svg` — SVG world map (55KB, CC BY-SA 3.0)

**Two-source data flow:**
1. App fetches `countries.json` + `wb-data.json` in parallel
2. Merges by `isoCode` — WB data under `country.wb` namespace
3. If `wb-data.json` fails, app degrades gracefully

**Composite scoring** (in RankingsView, reused by Scorecard):
- EPI 25% + Renewable Energy 20% + Forest 15% + Protected Areas 15% + Air Quality 15% + CO₂ Efficiency 10%
- Normalized 0-100, null values treated as 0
- Scorecard grades (A+ to F) computed by percentile ranking within the 80-country dataset

**countries.json schema:** `countryEn/Zh`, `agencyEn/Zh`, `descriptionEn/Zh`, `website`, `flagUrl`, `isoCode`, `region`, `established`, `responsibilities`, `contact`, `treaties`, `epiScore`, `netZeroTarget`, `keyLaws`, `data`, `parisAgreement` (NDC target + rating + history + nextDeadline), `montrealProtocol` (kigaliAmendment), `cbd` (30×30), `carbonPricing` (ETS/tax/price/coverage), `reportingStatus` (BTR).

**Dual-row filters:** compliance status (NDC/carbon price/BTR/Kigali/30×30) + responsibility tags. Both can stack.

**View modes:** Cards (paginated grid) | Rankings (sortable leaderboard table). Toggled via buttons.

**i18n:** `t(zh, en)` helper + `TREATY_LABELS` (17) + `RESPONSIBILITY_LABELS` (9) + `NDC_RATING_CONFIG` (7 levels).
